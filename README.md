# MySQL_Superheroes
 SQL and PHP based superhero networking


## CRUD elements
 - Create new heroes
 - Add new abilities from list to a hero's entry
 - Display hero information and connections
 - Update a hero's biography
 - Delete abilities from a hero's entry
